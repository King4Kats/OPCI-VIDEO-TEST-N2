========================================
DEPLOIEMENT SUR NOUVEAU PC
========================================

Etapes:
1. Installer Python 3.12 (cocher "Add to PATH")
2. Installer FFmpeg (ajouter au PATH)
3. Installer Ollama + modele (mistral)
4. Copier le projet
5. Creer venv: python -m venv venv_312
6. Activer: venv_312\Scripts\activate
7. Installer deps: pip install -r requirements.txt
8. Lancer: launch.bat

Temps d'installation: ~30-60 min
Espace requis: 10-30 GB

========================================
CONFIGURATION RECOMMANDEE
========================================

PC Minimum:
- Windows 10/11 64 bits
- CPU 4 coeurs
- RAM 8 GB
- Disque 30 GB

Modeles:
- Whisper: medium (defaut)
- Ollama: qwen3-coder:30b ou mistral

========================================
UTILISATION
========================================

1. Double-cliquer sur launch.bat
2. Selectionner fichiers video (MTS/MP4)
3. Cliquer "Demarrer l'analyse"
4. Attendre fin du traitement
5. Consulter les segments proposes
6. Previsualiser si necessaire
7. Choisir dossier de sortie
8. Exporter tous les segments
